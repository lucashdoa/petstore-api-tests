# petstore-api-tests

🤡 A demo API testing project using Jest.

System under test: [Swagger Petstore](petstore.swagger.io/v2)

---

## 🏃‍♀️ How to Run

1. Install dependencies

```
  npm install
```

2. Run all tests

```
  npm run test
```
